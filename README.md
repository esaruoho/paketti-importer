# paketti-importer
The Import features of Paketti, my quality-of-life-tool for Renoise.

# Supported Formats

## Importing
- .REX (ReCycle V1 Legacy format)
- .RX2 (ReCycle V2 format)
- .SF2 (Soundfont V2)
- .PTI (Polyend Tracker Instrument)
- more TBA
